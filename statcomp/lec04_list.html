List

List

  • Recursive vectors, instead of atomic
  • Foundations for data frame, objective-oirented programming
  • Indexed by number or name (tag)
  • Access elements by $ and [[]]

Indexing

Single element


                    lst$c                                                                     
                    lst[["c"]]
                    lst[[1]]

Sublist

                                                            
                    lst["c"]
                    lst[1]

Note lst[1:2] is ok but lst[[1:2]] is not

Adding and Deleting

Adding one or multiple elements

                                                            
                    z$c <- "sailing"
                    z[[3]] <-  "sailing"
                    z[4:5] <- c(TRUE, FALSE) # multiple elements

Deleting one element (note later indices -1 in new list

                                                            
                    z[[2]] <- NULL

Concatenating lists

                                                            
                    > c(list(1), list(TRUE))
                    [[1]]
                    [1] 1
                    
                    [[2]]
                    [1] TRUE
