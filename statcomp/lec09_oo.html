Objective Oriented Programming

  • Popular design and may lead to clearer and more usable code
  • R is not full OOP like JAVA, C++
  • R enables OOP through S3 (dominant) or S4 (e.g. Bioconductor)
  • Limited OOP but enough for most data tasks

R Theme

  • Object: everything is object
  • Encapsulation: package similar objects together in class
  • Polymorphic: same function name calls on different implementations depending on the data class
  • Inheritance: allows extending more general class to more specific ones

Stationary Distribution

  • Markov chain with stationary distribution $\pi$ and transition matrix $P$ on states
  • Stationary distribution means that $$\pi = \pi P \;\; \mbox{or} \;\; (I - P^T)\pi = 0$$ with the constraint $1^T \pi = 1$
  • To calculate $\pi$, solve the linear equations above. Note one row in $I-P$ is redundant and needs to be removed

Implementation


                    findpi1 <- function(p) {
                        n <- nrow(p)
                        imp <- diag(n) - t(p)   
                        imp[n,] <- rep(1,n)   
                        rhs <- c(rep(0,n-1),1)  
                        pivec <- solve(imp,rhs)  
                        return(pivec)
                     }
