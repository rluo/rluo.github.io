Factor

Factor

  • Used for nominal, categorical vars
  • Like vectors but usually with small set of dinstinct values, aka levels
  • Built on integer vector with predefined levels
  • May appear like character vector but technically different

Type


                    x <- factor(c("a", "b", "b", "a"))
                    x
                    #> [1] a b b a
                    #> Levels: a b                  
                    typeof(x)
                    #> [1] "integer"
                    attributes(x)
                    #> $levels
                    #> [1] "a" "b"
                    #> 
                    #> $class
                    #> [1] "factor"

Storage

Saves lots of memory, especially when level names are long

Factor Functions

  • tapply split vector into segments by factor levels and then apply function
  • split only does the splitting as in above
  • by apply to objects beyond vectors

Table

Table

  • Like data frame, buth with class table
  • Some technical differences in the next example

Example 6.3.2


                    subtable <- function(tbl,subnames) {
                        # get array of cell counts in tbl
                        tblarray <- unclass(tbl)   
                        # calling do.call() on the "[" function; we need to build up a list
                        dcargs <- list(tblarray)   
                        ndims <- length(subnames)  # number of dimensions 
                        for (i in 1:ndims) {
                           dcargs[[i+1]] <- subnames[[i]]
                        }   
                        subarray <- do.call("[",dcargs)   
                        dims <- lapply(subnames,length)  
                        subtbl <- array(subarray,dims,dimnames=subnames)
                        class(subtbl) <- "table"
                        return(subtbl)
                     }
